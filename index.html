Advanced — NCBI API key & rate-limit settings

Without a key NCBI allows ~3 requests/second; the tool stays under that and retries with backoff on a 429. A free key (NCBI account → API Key Management) lifts this to 10/s. Results cache in this browser for 24 h. Key, email and theme are stored only locally.

Gene model — exon / intron structure

Sources: NCBI ClinVar (aggregate germline classification), UniProt (reviewed human entry), Ensembl (canonical transcript). Counts reflect aggregate VCV classification. Residue positions come from HGVS p. where present; splice/intronic variants are placed at an approximate residue (cDNA ÷ 3) and flagged. Whole-gene CNVs are counted but not plotted. The aligned exon strip maps coding sequence to the residue axis; the detailed gene model is drawn in genomic coordinates (its own scale). Custom mutations are user-supplied and not from ClinVar. ClinVar updates weekly. Exploratory tool — not for clinical use.
